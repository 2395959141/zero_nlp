# zero to nlp

## 特点
1. 注重中文的nlp建设
2. 基于transformers库
3. 涉及文本清洗、训练、部署等内容
4. 提供公开的数据
5. 开箱即用，保证代码都是可跑通，替换数据即可训练出自己的模型
6. transformers库的源码级介绍，
7. 每一个项目，都具有数据、模型训练、部署等模块
8. 基于pytorch，不用tensorflow
9. 模型方面，只介绍大模型，不再介绍传统文本模型
10. 每一个项目，都会给出相关的注意事项
